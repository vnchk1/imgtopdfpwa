# Изображения в PDF - PWA

Веб-приложение для конвертации изображений в один PDF файл. Работает на десктопе и мобильных устройствах (Android).

## Возможности

- ✅ Поддержка форматов: JPEG, PNG, HEIF/HEIC
- ✅ Работает в Google Chrome на всех платформах
- ✅ PWA - работает офлайн после первой загрузки
- ✅ Отображение статуса обработки каждого изображения в реальном времени
- ✅ Сортировка по дате или алфавиту
- ✅ Выбор ориентации страницы
- ✅ Оптимальное масштабирование изображений без искажений
- ✅ **Безопасная работа с файлами** - оригинальные файлы не изменяются и не удаляются

## Платформы

Приложение разделено на две отдельные версии для оптимальной работы на разных устройствах:

### Десктопная версия (`index.html`)
- Поддержка drag & drop (перетаскивание файлов)
- Выбор файлов через стандартный диалог
- Работа с File объектами напрямую
- Полная функциональность

### Мобильная версия (`mobile.html`)
- Отдельная страница для мобильных устройств
- Выбор изображений из галереи
- Выбор изображений через файловый менеджер
- Оптимизированная обработка для мобильных устройств
- **Безопасность**: все файлы копируются в память как Blob, оригиналы не затрагиваются
- Ссылка на мобильную версию доступна на главной странице

## Безопасность файлов

⚠️ **Важно**: Приложение создает безопасные копии выбранных файлов в памяти браузера. Оригинальные файлы на вашем устройстве:
- **НЕ изменяются**
- **НЕ удаляются**
- **НЕ повреждаются**

Все операции выполняются только с копиями файлов в памяти.

## Развертывание на Netlify

1. Загрузите все файлы в репозиторий Git
2. Подключите репозиторий к Netlify
3. Настройки сборки не требуются (статический сайт)
4. Приложение готово к использованию!

## Локальная разработка

Откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python
python -m http.server 8000

# Node.js
npx serve
```

## Использование

### На компьютере
1. Откройте приложение в браузере
2. Выберите изображения через кнопку "Выбрать изображения" или перетащите их в окно браузера
3. Настройте сортировку и ориентацию страницы
4. Нажмите "Конвертировать в PDF"
5. Дождитесь завершения обработки (прогресс отображается в реальном времени)
6. PDF файл автоматически скачается

### На Android
1. Откройте приложение в браузере Chrome
2. Перейдите на мобильную версию по ссылке на главной странице (или откройте `mobile.html` напрямую)
3. Нажмите "Выбрать изображения"
4. Выберите источник: Галерея или Файловый менеджер
5. Выберите одно или несколько изображений
6. Настройте сортировку и ориентацию страницы
7. Нажмите "Конвертировать в PDF"
8. Дождитесь завершения обработки (прогресс отображается в реальном времени)
9. PDF файл автоматически скачается

## Используемые библиотеки

- **jsPDF** - генерация PDF файлов
- **heic2any** - конвертация HEIF/HEIC в JPEG
