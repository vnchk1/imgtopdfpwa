# Изображения в PDF - PWA

Веб-приложение для конвертации изображений в один PDF файл.

## Возможности

- ✅ Поддержка форматов: JPEG, PNG, HEIF/HEIC
- ✅ Работает в Google Chrome на всех платформах
- ✅ PWA - работает офлайн после первой загрузки
- ✅ Отображение статуса обработки каждого изображения
- ✅ Сортировка по дате или алфавиту
- ✅ Выбор ориентации страницы
- ✅ Сохранение в выбранную директорию (через File System Access API)
- ✅ Оптимальное масштабирование изображений без искажений

## Создание иконок

Перед развертыванием необходимо создать иконки для PWA:

1. Откройте `generate-icons.html` в браузере
2. Нажмите кнопку "Сгенерировать иконки"
3. Скачайте созданные файлы `icon-192.png` и `icon-512.png`
4. Поместите их в корневую директорию проекта

Альтернативно, если у вас установлен Node.js:
```bash
npm install canvas
node create-icons.js
```

## Развертывание на Netlify

1. Создайте иконки (см. выше)
2. Загрузите все файлы в репозиторий Git
3. Подключите репозиторий к Netlify
4. Настройки сборки не требуются (статический сайт)
5. Приложение готово к использованию!

## Локальная разработка

Просто откройте `index.html` в браузере или используйте локальный сервер:

```bash
# Python
python -m http.server 8000

# Node.js
npx serve
```

## Используемые библиотеки

- **jsPDF** - генерация PDF файлов
- **heic2any** - конвертация HEIF/HEIC в JPEG

## Примечания

- Для работы с HEIF/HEIC требуется подключение к интернету (библиотека загружается из CDN)
- File System Access API поддерживается в Chrome/Edge. В других браузерах файл будет скачан автоматически
- PWA работает офлайн после первой загрузки
